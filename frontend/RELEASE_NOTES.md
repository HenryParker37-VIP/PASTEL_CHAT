# PastelChat Desktop - Release Notes

## Version 1.0.0

**Release Date:** 2024

### What's New
- Initial Windows desktop release
- Full chat functionality with real-time messaging
- Google OAuth login
- Push notification support
- Auto-update system (checks for updates on startup)

### System Requirements
- Windows 7 / 8 / 10 / 11 (64-bit)
- 200 MB disk space
- Internet connection required

### Downloads
- `PastelChat Setup 1.0.0.exe` — Installer (recommended)
- `PastelChat 1.0.0.exe` — Portable (no installation needed)
